# Robot Assistant Simulator

This is a simple Python-based robot assistant simulator that interacts with the user based on robot name, battery level, and operational mode.

---

## What the Project Does

- Asks the user for:
  - Robot name
  - Battery level (0â€“100)
  - Operation mode (Idle, Active, Shutdown)
- Responds with:
  - Battery status: low, okay, or fully charged
  - Robot status: waiting, working, or shutting down

---

## How to Run

1. Make sure you have Python installed (version 3+)
2. Download or clone this repository
3. Open a terminal or command prompt in the project folder
4. Run the following command:
